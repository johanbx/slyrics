# slyrics
A Spotify Lyrics Viewer, requires Spotify on windows

Check releases for newest version: https://github.com/johanbx/slyrics/releases

1. Download the .exe file
2. Launch the .exe file
3. Launch Spotify
4. Enjoy your lyrics :)
